Método de Bisección

Divide el intervalo [xi, xf] a la mitad en cada iteración.
Requiere que f(xi)·f(xf) < 0 — si no, el intervalo se expande automáticamente.

xm = (xi + xf) / 2
Si f(xi)·f(xm) > 0 → xi = xm  |  Si f(xi)·f(xm) < 0 → xf = xm  |  error = (xm − xi) / xm × 100
